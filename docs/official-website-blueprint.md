# XXX科技有限公司官网整体设计

## 建设目标

XXX科技官网面向客户、合作伙伴、投资者、求职者和媒体公众，核心目标是展示公司可信度、行业解决方案、产品与服务能力、投资者关系、新闻动态和招聘入口。

## 信息架构

一级导航建议：

- 首页
- 关于XXX科技
- 新闻中心
- 解决方案
- 产品与服务
- 投资者关系
- 加入我们
- 联系我们

二级导航建议：

- 关于XXX科技：公司简介、发展历程、企业文化、资质荣誉、社会责任、联系我们
- 新闻中心：公司新闻、行业动态、媒体报道、公告通知
- 解决方案：保险、银行、证券、政府、医疗、教育、能源、企业数字化
- 产品与服务：软件开发、系统集成、运维服务、云服务、大数据、人工智能、信息安全
- 投资者关系：公司公告、定期报告、股票信息、治理结构、投资者互动
- 加入我们：社会招聘、校园招聘、员工发展、招聘流程

## 角色与权限

- 游客：浏览公开页面、搜索内容、提交联系表单、投递简历。
- 内容编辑：维护新闻、案例、产品、解决方案、招聘岗位。
- 内容审核：审核发布内容、管理上下架。
- 投资者关系管理员：维护公告、财报、股票资料、治理信息。
- 系统管理员：管理用户、角色、权限、站点配置、操作日志。

## 核心业务闭环

1. 内容生产：编辑创建新闻、案例、产品、解决方案。
2. 内容审核：审核员检查内容合法性、准确性、展示位置。
3. 内容发布：内容按频道、标签、推荐位发布到官网。
4. 用户触达：访客通过导航、搜索、相关推荐浏览内容。
5. 线索转化：访客提交咨询、合作、招聘或投资者互动表单。
6. 后台跟进：管理员查看线索、分派负责人、标记处理状态。

## 非功能要求

- 响应式：支持桌面、平板、移动端。
- SEO：支持自定义标题、描述、关键词、结构化数据、站点地图。
- 性能：静态内容 CDN 化，列表分页，图片懒加载。
- 安全：后台鉴权、RBAC、操作审计、上传文件校验、接口限流。
- 合规：ICP备案、公示信息、隐私政策、版权声明、投资者资料留痕。
- 可维护：内容模型清晰，前后台共享枚举，接口版本化。

## 推荐技术栈

- 前台：Next.js / Nuxt / Vite + React 或 Vue。
- 后台：React Admin / Vue + Element Plus / Ant Design。
- 服务端：NestJS / Spring Boot / Django REST Framework。
- 数据库：PostgreSQL / MySQL。
- 缓存：Redis。
- 搜索：Meilisearch / Elasticsearch / 数据库全文索引。
- 文件存储：对象存储 OSS / S3 兼容存储。
