# 前台设计

## 设计原则

- 风格：稳健、专业、科技感、企业可信度。
- 信息密度：官网首页突出业务能力，详情页突出可读性和转化。
- 色彩：以企业蓝、深灰、白色为主，辅以高亮科技蓝或绿色状态色。
- 交互：导航清晰、搜索可达、移动端首屏简洁。

## 首页

模块建议：

- 顶部导航：Logo、主导航、搜索、语言切换。
- 首屏 Banner：公司定位、核心能力、重点业务入口。
- 公司概览：成立背景、服务行业、客户数量、研发能力等关键数字。
- 解决方案：保险、银行、政府、医疗等重点行业卡片。
- 产品与服务：软件开发、系统集成、运维服务、云与数据智能。
- 新闻动态：公司新闻、行业动态、媒体报道。
- 投资者关系入口：公告、报告、股票信息。
- 招聘入口：社会招聘、校园招聘。
- 页脚：联系方式、友情链接、版权、备案号。

## 关于XXX科技

- 公司简介。
- 发展历程时间轴。
- 企业文化。
- 资质荣誉。
- 社会责任。
- 联系方式与地图。

## 新闻中心

- 新闻列表：分类、年份筛选、关键词搜索。
- 新闻详情：标题、发布日期、来源、正文、上一篇/下一篇、相关推荐。

## 解决方案

- 方案列表：按行业筛选。
- 方案详情：
  - 行业背景。
  - 客户痛点。
  - 解决方案架构。
  - 核心能力。
  - 客户价值。
  - 相关产品。
  - 相关案例。
  - 咨询入口。

## 产品与服务

- 产品/服务列表。
- 详情页：
  - 产品简介。
  - 功能特性。
  - 适用场景。
  - 技术优势。
  - 资料下载。
  - 联系咨询。

## 投资者关系

- 投资者关系首页：最新公告、定期报告、公司治理、股票信息入口。
- 公告列表：按年份、类型筛选。
- 文件详情或下载：PDF 在线预览/下载。
- 投资者互动：联系方式和留言表单。

## 加入我们

- 招聘首页：雇主品牌、成长体系、招聘类型入口。
- 岗位列表：职位类型、地点、关键词筛选。
- 岗位详情：职责、要求、地点、投递按钮。
- 简历投递表单：基础信息、附件上传、隐私授权。

## 搜索页

- 支持全站搜索新闻、解决方案、产品、案例、公告、招聘。
- 搜索结果显示类型、标题、摘要、发布时间。
- 支持分页。

## 移动端适配

- 顶部导航折叠为抽屉菜单。
- 首页模块按纵向顺序展示。
- 表格类公告列表改为卡片列表。
- 表单字段单列展示。
