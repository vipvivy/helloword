# 后台管理设计

## 后台首页

- 今日新增线索、待审核内容、已发布内容、招聘投递数。
- 快捷入口：新建新闻、新建解决方案、新建招聘岗位、上传投资者公告。
- 最近操作日志和异常登录提醒。

## 内容管理

模块：

- 页面管理：公司简介、企业文化、社会责任、联系我们等。
- 新闻管理：公司新闻、行业动态、媒体报道、公告通知。
- 解决方案管理：按行业分类维护方案内容。
- 产品与服务管理：维护产品特性、适用场景、资料下载。
- 客户案例管理：维护案例内容与关联方案。
- Banner 与推荐位：维护首页轮播、首页重点解决方案、首页新闻推荐。

核心功能：

- 草稿、提交审核、发布、下架、归档。
- 富文本编辑、图片上传、文件上传。
- SEO 字段配置。
- 预览链接。
- 定时发布。

## 投资者关系管理

- 公告文件上传。
- 定期报告维护。
- 公司治理文件维护。
- 股票信息链接配置。
- 披露日期和年份筛选。
- 支持外部交易所原文链接，便于合规追溯。

## 招聘管理

- 岗位发布、关闭和排序。
- 社招、校招、实习分类。
- 简历附件查看和下载。
- 投递状态流转：新投递、筛选中、面试中、已录用、已拒绝。

## 线索管理

- 联系我们、合作咨询、投资者互动、简历投递统一进入线索池。
- 支持状态、来源、时间、关键词筛选。
- 支持分派负责人、备注跟进记录。
- 支持垃圾线索标记和导出。

## 系统管理

- 用户管理。
- 角色权限。
- 菜单管理。
- 站点配置。
- 文件资源管理。
- 操作日志。
- 登录日志。

## 权限设计

权限按资源和动作拆分：

- `article:list`
- `article:create`
- `article:update`
- `article:review`
- `article:publish`
- `article:delete`
- `investor_document:publish`
- `job:update`
- `lead:assign`
- `system:user_manage`

## 审核流

推荐初期使用简单审核流：

1. 编辑保存草稿。
2. 编辑提交审核。
3. 审核员通过后发布，或退回修改。
4. 已发布内容可申请下架或直接归档。

后续可扩展多级审核、敏感词检测和法务/合规确认节点。
